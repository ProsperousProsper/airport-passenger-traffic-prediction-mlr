2.   Installing Mavis Beacon Teaches Typing

Mavis Beacon Teaches Typing installs onto your computer’s hard drive, so you will need to have at least 300MB of free space on your hard drive to complete the installation. For best results, you should close all other programs and applications before starting the installation.  Follow the instructions below to install the program. There are separate instructions for installing on Windows® and Macintosh®operating systems. Please note that the following instructions assume that the AutoPlay feature on your CD-ROM is active.

 

Windows®

1.    Insert the Mavis Beacon Teaches Typing CD into your computer’s CD-ROM drive.

2.    Before you can complete the installation process you must accept the terms of the license agreement. You will be able to print the agreement for your records from this screen.  Once you have accepted the terms of the agreement, click Next to continue.

3.    At this point the installer will select a Destination Folder for the program.  Click Change if you would like to install the program to a different folder.  Click Next to continue the installation.

4.    Click Install to install the program.  Click Back to return to the previous screen. (You can click Cancel at any point during the installation to cancel the installation)

5.    Once the installation is complete, click Finish to exit the installer. (The setup program will place Mavis Beacon file icons in the Start menu, under the Broderbund program group)

6.    To run the program after installation, go to the Start menu and select Broderbund, click Mavis Beacon Teaches Typing 20, and Mavis Beacon Teaches Typing 20.

 

Macintosh® 

  1. Insert the Mavis Beacon Teaches Typing CD in the CD-ROM drive.
  2. Before you can complete the installation process you must accept the terms of the license agreement. You will be able to print the agreement for your records from this screen. Once you have accepted the terms of the agreement, click Agree to continue.
  3. Click Install to install the program. Click Quit to return to the previous screen.
  4. At this point you must select a Destination Folder for the program. We recommend selecting Applications from the list and clicking Choose.
  5. Once the installation is complete, click Quit to exit the installer.
  6. To start the program after the initial installation, double-click the program icon. The program icon will be located on the hard drive in:

·         Applications

·         Mavis Beacon Teaches Typing 20 folder

·   Mavis Beacon Teaches Typing 20 icon

 

Running Mavis Beacon Teaches Typing

On Windows:

1.             At the active Windows desktop, double-click the Mavis Beacon Teaches Typing icon.

2.             Or, click Start, choose Programs, select Broderbund folder, and then click Mavis Beacon Teaches Typing 20.

On the Macintosh:

1.     Locate the folder where the application was installed.

2.     In the folder, launch the Mavis Beacon Teaches Typing 20 icon.

 

4. Uninstalling Mavis Beacon Teaches Typing

      On Windows:

1.    Go to the Start menu and select Broderbund, click Mavis Beacon Teaches Typing 20, and click Uninstall Mavis Beacon Teaches Typing 20

2.    Then follow the onscreen instructions to finish the Uninstall process.

On Macintosh:

  1. Select the Mavis Beacon Teaches Typing program icon located on the hard drive in:
    • Applications

·   Mavis Beacon Teaches Typing folder

2.    Drag the program folder into the trash.

6. Getting Help with Mavis Beacon Teaches Typing

For more information on using Mavis Beacon Teaches Typing 20, please review the following information.

 

Mavis Beacon Teaches Typing 20 User Guide

After launching Mavis Beacon Teaches Typing, you can access the Mavis Beacon Teaches Typing 20 User Guide.  Click the Help button found on the Home screen of the program and click the link to the User Guide. You can also access the User Guide from the Help button located on the top left navigation bar found throughout the program.

On Windows, you can also access the User Guide from the Start Menu.  From the Start menu, select Broderbund, click Mavis Beacon Teaches Typing 20, and click View Mavis Beacon Teaches Typing User Guide.

On the Macintosh, locate the file User Guide.pdf in the Mavis Beacon Teaches Typing 20 application folder and launch it.

 

NOTE: Adobe ® Acrobat Reader is required to view and print the User Guide.  If you do not have Acrobat Reader installed on your computer, you can download a free copy from the Adobe website at http://www.adobe.com/.

 

Full Screen Mode

The program automatically runs in Full Screen mode. To switch to the smaller “Windowed mode,” please set your desktop resolution to 1152x864 or greater.
